  • Hal Jordan = Parallax

    This plugin was originally written for use with the main landing page of my startup, Infinite.ly, so this demo page uses many of the same graphical assets. My initial goal with writing HJ=P was to create a parallaxing system that uses z-indexes to judge the relative distance between objects, as this is already how web designers are layering their DOM elements on top of each other. It makes sense to leverage the CSS that's already there. Here's how to use HJ=P on your pages.

  • Add the script to your page, somewhere, wherever.

    The only requirement is jQuery itself, but you knew that already, didn't you? <script language="Javascript" src="/javascripts/jquery.js"></script> <script language="Javascript" src="/javascripts/jquery.hal.jordan.is.parallax.js"></script>

    Then it's just a matter of attaching $.parallax to your main container (most likely $('body')) inside a document.ready() block, thusly: $(function(){ $('body').parallax({ targets : ".parallax_target" }); });

    As you can probably guess, the targets parameter is a jQuery selector that enables the parallax effect for every element it matches. Any valid jQuery selector should work here.
    HJ=P works by looking at the individual z-indexes of all the target elements, and slowing down their scrolling speed based on this. Specifically: the lower a z-index is, the "further away" it is, therefore, the slower it should move. A target with a z-index of 2 will be 10x slower than a target with a z-index of 20.

    You can see the effect on this page's background, with the planets, asteroids and space background all moving at different speeds as the window is scrolled up and down. In this scenario, the furthest object is obviously the space background, with the asteroids being the closest. (Well, technically, the content you're reading is the closest, since that's on the highest z-index, but the parallax plugin doesn't touch that.)

    Check out other options.

  • Options

    Here's the parallax call again, but with all the options: $('body').parallax({ targets : ".parallax_target", orientation : "vertical", highest_z : 99, lowest_z : 1, force_height : 3500, force_width : 1000 });

    targets, possible : any valid jQuery selector, e.g., $(".content, .article")
    Just because you can use any valid jQuery selector, doesn't mean you necessarily have to be cute here. It's usually better to simply add a single class to all of the elements you want to apply the plugin to, and just select them that way.
    orientation, possible : "vertical", "horizontal"
    highest_z, possible : any integer
    lowest_z, possible : any integer
    force_height, possible : any integer, greater than your current container height
    force_width, possible : any integer, greater than your current container width
